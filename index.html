Ore GitHub Project Page

Client-side

Ore Data

A lightweight JavaScript utility library that simplifies working with JSON data and dynamic rendering in the browser. It provides helpers for parsing URL parameters, fetching and posting JSON via fetch, and rendering templates directly from objects or API responses.

Ore Tag

No build tools. No JSX. Ore Tag is a component library to build simple, lightweight components that run lightning-fast on the metal, directly in your browser. Effortlessly craft high-performance, elegant web components with zero setup, delivering smooth and seamless experiences that feel instant and intuitive.

Server-side

Ore DB

A lightweight, no-SQL JSON database engine for .NET. It uses in-memory storage with thread-safe access and simple commit-based persistence, making it ideal for small tools, prototypes, and applications that need structured data without a full database engine.
